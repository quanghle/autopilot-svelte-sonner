# Contributing to autopilot-svelte-sonner

Thank you for your interest in contributing! This project is a fork of [svelte-sonner](https://github.com/wobsoriano/svelte-sonner) by [Robert Soriano](https://github.com/wobsoriano). Feature requests that align with the original library are preferred.

## Getting Started

1. Fork and clone the repository
2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Start the development server:

   ```bash
   pnpm dev
   ```

## Development

- **Dev server:** `pnpm dev`
- **Build:** `pnpm build`
- **Run tests:** `pnpm test`
- **Unit tests:** `pnpm test:unit`
- **Lint:** `pnpm lint`
- **Format:** `pnpm format`
- **Type check:** `pnpm check`

## Submitting Changes

1. Create a new branch from `main` for your changes
2. Make your changes and ensure tests pass
3. Run `pnpm lint` and `pnpm check` before committing
4. Submit a pull request with a clear description of your changes

## Reporting Issues

Use the [issue templates](https://github.com/quanghle/autopilot-svelte-sonner/issues/new/choose) to report bugs, request features, or suggest documentation changes.
